# -Weather_Extension
A chrome Extension that fetches Weather Update.

## How to setup
1. Download Zip file and extract it.
2. Go to Chrome->Manage Extensions->Enable Developers Mode->Load Unpacked.
3. Now locate the folder.
4. Pin the Extention.
